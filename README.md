# 🏥 Hospital Management System

A modern **Hospital Management System (HMS)** developed to simplify hospital operations, improve patient management, and streamline administrative tasks. This web-based application provides an intuitive dashboard for managing patients, doctors, appointments, billing, laboratory reports, and more.

---

## 📌 Features

### 👨‍⚕️ Admin Module

* Secure Admin Login
* Dashboard with Statistics
* Manage Doctors
* Manage Patients
* Manage Staff
* Manage Departments
* Appointment Management
* Billing & Payments
* Laboratory Management
* Pharmacy Management
* Reports & Analytics

### 🩺 Doctor Module

* Doctor Login
* View Assigned Patients
* Manage Appointments
* Update Patient Diagnosis
* Prescribe Medicines
* View Medical History

### 👤 Patient Module

* Patient Registration & Login
* Book Appointments
* View Medical Records
* Download Lab Reports
* View Prescriptions
* Billing History

### 🧪 Laboratory Module

* Manage Lab Tests
* Upload Test Reports
* View Patient Test History

### 💊 Pharmacy Module

* Medicine Inventory
* Medicine Stock Management
* Prescription Verification

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/hospital-management-system.git
```

2. Move the project to the **htdocs** folder (XAMPP).

3. Start **Apache** and **MySQL** from XAMPP.

4. Import the database

* Open phpMyAdmin
* Create a database named `hospital_management`
* Import the `hospital.sql` file

5. Open your browser

```
http://localhost/hospital-management-system
```
